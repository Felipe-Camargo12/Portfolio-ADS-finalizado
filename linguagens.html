Linguagens de Programação - ADS

Linguagens de programação são instruções que permitem escrever programas de computador. Elas fornecem uma sintaxe/estrutura para expressar algoritmos e controlar o comportamento do computador. Existem siferentes linguagens para diferentes tipos de desenvolvimento de software.

Site de Receitas

Neste projeto, tivemos a oportunidade de explorar os elementos essenciais do HTML pela primeira vez. Aprendemos como estruturar e organizar o conteúdo em um site, utilizando tags básicas como o "h1". Além disso, descobrimos como criar listas ordenadas e não ordenadas para apresentar informações de maneira organizada.

Aprendemos também a adicionar imagens a uma página HTML, tornando-a visualmente mais atrativa. Outro aspecto que exploramos foi a criação de links que direcionam os usuários para outras páginas, permitindo uma navegação mais completa pelo site.

Currículo

Por meio desse projeto, aprofundamos nosso conhecimento em relação à formatação visual das páginas usando CSS. Foi uma oportunidade de integrar o CSS ao HTML e, assim, alcançar uma formatação mais abrangente e sofisticada para o site que estávamos desenvolvendo. Durante o projeto, aprendemos a utilizar elementos de cor e a aplicar formatações específicas às divisões (divs), o que nos permitiu organizar de maneira mais eficiente todos os elementos presentes no site. Além disso, utilizamos novamente o recurso de criação de links entre as páginas, proporcionando uma experiência de navegação completa e coesa aos usuários.

HTML - Semântico

Nesse projeto, adquiri conhecimentos sobre a relevância do HTML semântico e a sua utilização adequada. Compreendi a importância de escolher as tags apropriadas para cada tipo de conteúdo, garantindo uma estruturação clara e significativa do site. Além disso, aprendi a criar novos elementos e aplicar estilos, como sombras, para melhorar a aparência visual. Também explorei recursos avançados, como o uso do hover, que permite que determinados elementos mudem quando o cursor do mouse é posicionado sobre eles. Além disso, adquiri habilidades para criar botões interativos e caixas de texto, aprimorando a experiência do usuário e fornecendo interatividade ao site.

Folha de Pagamento

Neste projeto, desenvolvemos um site que tinha a função de calcular os descontos da folha de pagamento. Foi uma das primeiras vezes em que tivemos a oportunidade de utilizar o JavaScript em conjunto com o HTML. Aprendemos a integrar o JavaScript ao HTML, permitindo que as funcionalidades e a lógica do cálculo fossem implementadas no site.

Além disso, adquirimos conhecimentos sobre como manipular dados usando JavaScript, possibilitando a realização dos cálculos necessários para os descontos da folha de pagamento. Essa experiência nos proporcionou uma compreensão mais aprofundada da interação entre HTML e JavaScript, além de nos capacitar a criar sites mais dinâmicos e funcionais.


          

Conversor

Nesse projeto, exploramos a criação de uma página que continha campos de entrada para textos em HTML e JavaScript, além de executá-los com uma formatação CSS pré-existente. Durante o projeto, aprendemos a criar formulários que permitiam aos usuários inserir texto tanto em HTML quanto em JavaScript, proporcionando uma interatividade maior ao site. Além disso, compreendemos como utilizar a formatação CSS já pronta para estilizar e apresentar os elementos da página de forma visualmente agradável. Essa atividade nos permitiu praticar a integração harmoniosa entre HTML, JavaScript e CSS, fortalecendo nossas habilidades em desenvolvimento web.
