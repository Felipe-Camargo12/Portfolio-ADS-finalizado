Ciência de Dados - ADS

A ciência de dados analisa dados usando técnicas estatísticas e de programação para obter insights valiosos e tomar decisões estratégicas. É uma disciplina essencial em várias áreas, impulsionando a inovação e o sucesso das organizações.

Análise de dados TITANIC

Participamos de um projeto que combinou Ciência de Dados e Banco de Dados, com o objetivo de normalizar um banco de dados dos passageiros do Titanic. Utilizamos os conhecimentos adquiridos nessas disciplinas para organizar as informações de forma eficiente, eliminando redundâncias e garantindo a integridade dos dados. A integração desses conceitos nos permitiu aprimorar nossas habilidades e compreender a importância da estruturação correta dos dados para análises eficazes.

Estátisca

No curso, estudamos diversos temas relacionados a estatística e cálculo. Aprendemos sobre amostragem, incluindo técnicas aleatórias, sistemáticas e estratificadas. Exploramos medidas de tendência central, como média, moda, mediana, percentil e quartil. Também abordamos medidas de dispersão, como amplitude, desvio padrão e variância.

Estudamos o teste de hipóteses, com foco no cálculo do Qui-Quadrado. Na inferência, discutimos a distribuição binomial e a regressão logística. Por fim, nos aprofundamos em derivadas, abordando a definição formal, retas tangentes, regras de derivação e comportamento das funções. Esses conceitos forneceram uma base sólida em estatística e cálculo, permitindo-nos analisar e interpretar dados de forma mais precisa e fundamentada.

Python

Python é amplamente utilizado na análise de dados devido à sua popularidade e às bibliotecas específicas disponíveis. Ao aprender Python, fomos capazes de utilizar bibliotecas como Pandas, Numpy e Matplotlib para executar os cálculos mencionados anteriormente. Essas bibliotecas nos permitiram extrair dados de tabelas, realizar cálculos estatísticos e criar gráficos de forma eficiente. Com essa abordagem, a análise de dados tornou-se mais fácil e acessível, fornecendo uma plataforma poderosa para explorar e interpretar conjuntos de dados complexos.

Análise de dados AVENGERS

O projeto acadêmico de ciência e análise de dados utiliza o banco de dados dos personagens da Marvel e o Google Colab com Python. Através da limpeza, transformação e visualização dos dados, incluindo a plotagem de gráficos, o projeto busca explorar e comunicar insights sobre os personagens de forma prática e atrativa. A análise dos dados permite identificar padrões e personagens com mais aparições ou informações, oferecendo uma oportunidade de aplicar conceitos teóricos e desenvolver habilidades de análise de dados.
